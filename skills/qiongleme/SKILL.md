# 穷了么 - 财务健康检测器

## 概述

一键检测用户离破产还有多远，生成毒舌诊断报告。

---

## 💀 核心流程

```
用户输入收入/支出 → 计算财务指标 → 生成毒舌诊断 → 提供抢救方案
```

---

## 📊 财务指标计算

### 输入数据（通过对话收集）
- 月收入（税后）
- 固定支出（房租/房贷、水电、保险等）
- 变动支出（餐饮、交通、娱乐、购物等）
- 储蓄余额
- 负债情况（信用卡、贷款等）

### 核心指标

| 指标 | 计算公式 | 危险阈值 |
|------|---------|---------|
| **生存天数** | 储蓄 / (月支出/30) | < 30天为危险 |
| **收支比** | 月支出 / 月收入 | > 80%为危险 |
| **债务率** | 月还款 / 月收入 | > 40%为危险 |
| **财务自由度** | 被动收入 / 总支出 | < 10%为危险 |

### 综合评分（0-100分）
```
基础分 = 50
+ 储蓄充足度 * 20
+ 收支健康度 * 15  
+ 债务安全度 * 15
- 消费成瘾扣分
```

---

## 🎭 诊断等级

| 分数 | 等级 | 状态描述 |
|------|------|---------|
| 80-100 | 🟢 财务自由 | "你已经赢了" |
| 60-79 | 🟡 亚健康 | "还能抢救一下" |
| 40-59 | 🟠 重症监护 | "危在旦夕" |
| 20-39 | 🔴 临床死亡 | "已出现尸斑" |
| 0-19 | ⚫ 彻底凉凉 | "建议重开" |

## 💬 毒舌文案库

### 开场白（随机选择）
- "欢迎来到'穷了么'，今天又是谁要面对现实了呢？"
- "准备好接受命运的审判了吗？"
- "我劝你深呼吸三次再往下看..."

### 根据消费习惯吐槽

**奶茶党**
- "本月奶茶支出：¥XXX。你知道这够买多少斤大米吗？"
- "你的血管里流淌的不是血，是珍珠奶茶"

**订阅狂魔**
- "你有17个自动续费，但每天只打开2个"
- "Netflix、Spotify、爱奇艺...你在养活整个互联网"

**外卖依赖症**
- "厨房对你来说只是个传说对吧？"
- "外卖小哥比你家人还熟悉你家门牌号"

**冲动消费型**
- "'我就看看不买'——本年度最大谎言"
- "你的购物车比你的存款还满"

**月光族**
- "工资到账→花呗还款→余额归零，用时3秒"
- "你的钱包比你的脸还干净"

## 🏥 抢救方案生成

根据具体问题提供3-5条可执行建议：

**通用模板**：
1. **立即止血** - 取消不必要的订阅/会员
2. **开源节流** - 具体可节省的金额和方式
3. **债务优化** - 如有负债，提供还款策略
4. **强制储蓄** - 设置自动转账到储蓄账户
5. **心理建设** - 毒鸡汤鼓励

## 📱 使用流程

### Step 1: 引导输入
```
💀 欢迎进行财务尸检

请告诉我：
1. 你每月税后收入是多少？
2. 固定支出（房租/房贷/保险等）：
3. 变动支出（餐饮/交通/娱乐等）：
4. 目前储蓄余额：
5. 每月需要还的债务（如有）：

（可以只回答知道的，我会基于现有信息分析）
```

### Step 2: 计算分析
- 实时计算各项指标
- 匹配对应的毒舌文案
- 生成个性化诊断

### Step 3: 输出报告
```
═══════════════════════════
    💀 穷了么 · 诊断报告
═══════════════════════════

患者：[昵称]
诊断日期：[日期]

【生命体征】
生存天数：XX天
收支比例：XX%
债务负担：XX%

【诊断结果】
🟠 等级：重症监护
死因预判：奶茶成瘾 + 冲动消费

【毒舌点评】
[个性化吐槽]

【抢救方案】
1. [具体建议]
2. [具体建议]
3. [具体建议]

预计续命：XX天

═══════════════════════════
[分享我的遗照] [再测一次]
═══════════════════════════
```
