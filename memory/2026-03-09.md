## 💡 经验教训

### 技术方面
1. **依赖管理**: npm 全局安装和本地安装要区分清楚
2. **TypeScript 类型**: 新增字段要及时更新类型定义
3. **RPC 节点**: 公共节点不稳定，需要多准备几个备选
4. **私钥安全**: 永远不要硬编码，使用环境变量

### 项目管理
1. **版本控制**: GitHub 提交信息要清晰（feat/fix/chore）
2. **部署流程**: 本地编译 → 推送到 GitHub → VPS 拉取/复制 → 重启服务
3. **错误处理**: 服务要有自动重启机制（systemd）
