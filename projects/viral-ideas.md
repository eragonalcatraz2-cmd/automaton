# 病毒式传播项目创意 - "死了么"风格

## 🎯 核心要素分析（从"死了么"学习）

| 要素 | "死了么"的做法 | 可复制性 |
|------|---------------|---------|
| **反常识概念** | 检测"死没死"这个荒谬问题 | ✅ 找生活中的荒诞点 |
| **极简交互** | 一键点击，立即出结果 | ✅ 必须3秒内完成 |
| **社交货币** | 截图分享很搞笑 | ✅ 结果要值得晒 |
| **零门槛** | 不用注册、不用思考 | ✅ 打开即用 |
| **话题性** | 死亡话题自带讨论度 | ✅ 触碰敏感但无害的边界 |

## 💡 我的创意提案

### 方案1: **"穷了么"** - 财务健康检测器 ⭐推荐

**一句话描述**: 一键检测你离破产还有多远

**使用场景**:
- 输入月收入和固定支出
- AI分析你的"财务死亡倒计时"
- 生成毒舌报告 + 自救建议

**示例输出**:
```
💀 检测结果：你已"财务死亡"

距离彻底破产：23天
死因：奶茶成瘾 + 冲动消费
遗言："我就看看不买"

🏥 抢救方案：
1. 立刻取消3个订阅服务
2. 本周不喝奶茶可续命7天
3. ...

[生成我的遗照] [分享到朋友圈]
```

**传播点**:
- 年轻人自嘲文化
- 毒舌AI吐槽很搞笑
- 结果值得截图分享
- 可以PK谁的"死期"更近

**我能做的**:
- ✅ 对话交互
- ✅ 数据分析和计算
- ✅ 毒舌文案生成
- ✅ 图片生成（需要创建技能）

### 方案2: **"秃了么"** - 脱发风险评估

**一句话描述**: 上传头顶照片，AI预测你的秃头时间线

**使用场景**:
- 拍照上传头顶
- AI分析发际线、发量密度
- 生成"秃头倒计时"和植发建议

**传播点**:
- 程序员/互联网人痛点
- 可以和朋友PK谁更危险
- "秃头预警"截图很好笑

**技术需求**:
- ❌ 图像识别（发际线检测）- 较难

### 方案3: **"卷了么"** - 内卷程度测试器

**一句话描述**: 测测你在职场/生活中有多卷

**使用场景**:
- 回答5-8个简单问题
- 生成"内卷指数"和类型标签
- 比如："卷王型"、"躺平型"、"假装努力型"

**示例输出**:
```
🌀 内卷指数：87/100

你是：【表面佛系暗中较劲型】
特征：
- 嘴上说着"无所谓"
- 背地里加班到11点
- 看到同事升职会失眠

同病相怜：全国有230万人和你一样

[查看我的卷友] [生成卷王证书]
```

**传播点**:
- 职场共鸣
- 类型标签适合分享
- 可以@同事互相伤害

**我能做的**:
- ✅ 问卷逻辑
- ✅ 结果生成
- ✅ 文案创作
- ✅ 数据统计

### 方案4: **"社死了么"** - 社死事件记录仪

**一句话描述**: 记录和评分你的社死瞬间，看看能不能上"社死排行榜"

**使用场景**:
- 匿名提交社死故事
- AI打分（1-10级社死）
- 生成"社死证书"
- 可选：投稿到每日精选

**传播点**:
- 八卦是人类天性
- 别人的社死最好笑
- 可以投票"这有多惨"

### 方案5: **"鸽了么"** - 放鸽子风险评估

**一句话描述**: 测测你今天放朋友鸽子的概率

**使用场景**:
- 输入今天的计划、心情、天气
- AI计算"鸽率"
- 如果超过80%，自动生成道歉文案

**示例输出**:
```
🐦 今日鸽率：92%

高危信号：
- 昨晚熬夜 ✓
- 今天下雨 ✓  
- 地点距离>5km ✓

预计借口："突然有点不舒服"
可信度：⭐⭐ (太低了，换个吧)

💬 AI推荐道歉文案：
"宝，我...（此处省略500字真诚检讨）"

[一键发送给朋友] [生成请假条]
```
